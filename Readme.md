# PipeDream

## Descripción
Repositorio con el código fuente y recursos del proyecto presentado en el hackathon. Contiene un prototipo utilizable, documentación mínima y scripts para ejecutar el proyecto localmente.

## Tecnologías
- Lenguajes: Python / JavaScript (ajustar según el contenido real)
- Librerías: (p. ej. scikit-learn, TensorFlow, Flask, React) — actualizar según el stack
- Entorno: Windows / Linux / macOS

## Requisitos
- Python 3.8+
- Node.js 14+
- pip / npm para gestión de paquetes
- Acceso a internet para descargar dependencias
- Docker (para lanzar backend)

## Instalación rápida
1. Clonar el repositorio:
    git clone <URL-del-repositorio>
2. Entrar al directorio:
    cd Hackathon_indesIa
3. Ejecutar backend:
    cd api
    docker compose up --build
3. Alternativa de ejecución del backend
    .\start-backend.ps1
4. Ejecutar frontend:
    .\start-frontend.ps1
